# Expense Tracker App

A React Native Expense Tracker application built using Expo and TypeScript.

## Features

- Splash Screen
- Dashboard
- Add Transaction
- Edit Transaction
- Delete Transaction
- Transaction List
- Transaction Details
- Search Transactions
- Filter by Income & Expense
- Summary Screen
- AsyncStorage Persistence

## Tech Stack

- React Native
- Expo
- TypeScript
- React Navigation
- Context API
- AsyncStorage
- DateTimePicker
- React Native Picker

## Installation

```bash
git clone <https://github.com/ksaket572-blip/expense-tracker-react-native>

cd ExpenseTracker

npm install

npx expo start
```
